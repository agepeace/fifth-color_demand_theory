# Fifth-Color Demand Theory (FCD Theory)

> A Structural Theory of Planar Graph Coloring

---

## Project Vision

The purpose of this project is **NOT** to produce another proof of the Four Color Theorem.

Instead, this project aims to establish a new mathematical framework for understanding

- the origin,
- propagation,
- interaction,
- and elimination

of the **Fifth-Color Demand** in planar graph coloring.

The Four Color Theorem is regarded as one possible application of this theory rather than its primary objective.

# Guiding Principles

This project follows several strict principles.

## Principle 1

Definitions come before theorems.

No conjecture may be used before all related objects are rigorously defined.

---

## Principle 2

Every statement belongs to exactly one category.

- Definition
- Proposition
- Lemma
- Theorem
- Corollary
- Conjecture
- Observation
- Open Problem

Mixing these categories is prohibited.

---

## Principle 3

Never confuse intuition with proof.

Research Notes are welcome.

Proofs must remain rigorous.

---

## Principle 4

Backward compatibility.

Once a notation is officially released,

future chapters should not redefine it.

---

## Principle 5

Every chapter should be independently readable.
